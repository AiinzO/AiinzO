## 🏆 Featured Project: PowerPong (Award Winner)
During a university course, my team and I developed **PowerPong** – an enhanced Pong clone featuring dynamic power-ups, multiple game modes (Classic, Survival, vs. AI), and a custom 2D physics engine. This project was honored with **two awards** in an internal university competition.

**Technical Highlights:**
- **Technologies:** Java, JavaFX, JPMS (Java Module System), ServiceLoader, `javax.sound`
- **Architecture:** Clean MVVM (Model-View-ViewModel) architecture. Strictly decoupled components using Facades and the Observer Pattern (custom Event/Listener system for game events).
- **Core Engineering:** Built a custom PhysicsEngine for precise AABB collisions and a dedicated Game Loop ensuring a stable 60 FPS performance.
- **AI:** Implemented a configurable AI with dynamic difficulty scaling (trajectory prediction & randomized human-like errors).
- **Testing:** Comprehensive Unit Testing for the physics engine, core game logic, and power-up states.

## ☀️ Featured Web Project: Solar-ROI Calculator
A modern, responsive web application built to calculate the exact return on investment for balcony solar power plants (Balkonkraftwerke). It combines a scientific simulation engine with a highly polished, conversion-optimized user interface.

**Technical Highlights:**
- **Technologies:** React 19, TypeScript, Tailwind CSS 4.0, Vite, Framer Motion, Recharts.
- **Architecture:** Engineered a custom `useSolarSimulation` hook acting as the core engine, cleanly separated from UI components. Implemented a DSGVO-compliant (GDPR) cookie consent manager from scratch.
- **Performance & UX:** Optimized for Core Web Vitals (CLS safe ad-containers), lazy loading for heavy chart bundles, and fluid glassmorphism animations.
- **Live Project:** [balkon-spar-rechner.de](https://balkon-spar-rechner.de) *(Private Repository)*
